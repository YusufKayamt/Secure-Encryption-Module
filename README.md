# Python Secure Cipher Tool

Bilgi Güvenliği dersi için hazırlanmış nesne yönelimli (OOP) şifreleme aracıdır.

## Teknik Detaylar
- **Algoritma:** Random Substitution Cipher
- **Veri Yapısı:** Hash Map (Dictionary) tabanlı hızlı eşleştirme.
- **Anahtar Güvenliği:** Anahtarlar Base64 formatında `.key` dosyasında tutulur.

## Kullanım
`python3 secure_cipher.py` komutu ile başlatılır.
